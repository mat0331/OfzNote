データ管理

ヘルプ

ショートカットキー

Ctrl + N 新規メモ作成
Ctrl + S 手動保存
Ctrl + F 検索
Ctrl + E エクスポート
Ctrl + B サイドバー開閉
Ctrl + , 設定を開く
F1 ヘルプを開く

Mac では Ctrl を Cmd に読み替えてください

主な機能

  • 自動保存: 3秒ごとに自動で保存されます
  • お気に入り: 星アイコンで重要なメモをマーク
  • タグ: メモに複数のタグを付けて分類（エディタ下部で入力）
  • フォルダ: メモをフォルダで整理（サイドバーで管理）
  • フォント調整: ヘッダーのドロップダウンで簡単に変更
  • 正規表現検索: 高度な検索・置換機能
  • ローカル保存: txtファイルとしてPCに保存可能

データについて

  • メモはブラウザの IndexedDB に保存されます
  • ブラウザを閉じてもデータは残ります
  • プライベートモードでは終了時に削除されます
  • 定期的にエクスポートしてバックアップすることを推奨
  • ローカルフォルダ保存を有効にすると、txtファイルとして保存できます

カスタマイズ

  • 3つのプリセットテーマ（ライト/ダーク/ブルー）
  • 高度なテーマエディタで11色をカスタマイズ可能
  • フォント、サイズ、行間を調整可能

トラブルシューティング

  • 表示がおかしい → Ctrl + Shift + R でスーパーリロード
  • データが消えた → ブラウザのデータを削除していないか確認
  • 保存されない → プライベートモードを使用していないか確認
  • ローカル保存が使えない → Chrome/Edge/Opera をご利用ください
